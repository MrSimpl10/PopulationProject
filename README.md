# PopulationProject
Project for EGME 202. Use existing population data of a certain group of people to create a function to estimate future populations. 

Our groups project is looking at US state populations over the last 5 years. Our dataset is from the data.census.gov website and has provided an estimate for the population of every individual state from 2020 to 2025. Our goal is to analyze the data to see if there's over arching trends over the years.
